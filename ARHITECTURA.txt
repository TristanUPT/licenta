1. STACK FINAL
================================================================================

FRONTEND:
- React 18 + TypeScript + Vite
- Zustand (state management)
- TailwindCSS + Radix UI (design system)
- react-knob-headless (knobs audio)
- React Flow (synth lab - v2/future)

VIZUALIZĂRI:
- wavesurfer.js v7 (waveform display + regions)
- audiomotion-analyzer (spectrum analyzer real-time)
- Canvas 2D custom (spectrogram, meters, gain reduction)
- SVG (curbe EQ, filter response)

AUDIO ENGINE:
- Rust → WebAssembly (DSP core - efecte, procesare)
  - fundsp (synthesis, filters, effects DSP)
  - rustfft / realfft (spectral analysis, FFT)
  - dasp (sample format conversion, buffers)
  - rubato (resampling)
  - hound (WAV decode/encode)
- Web Audio API (I/O, routing, AudioContext)
- AudioWorklet (bridge JS ↔ WASM)
- Tone.js (transport, scheduling, clock)

PERSISTENȚĂ:
- IndexedDB via idb (presets, audio files, snapshots)
- localStorage (settings, UI preferences)

AUDIO SAMPLES:
- Freesound.org CC0 curated set (~20-30 samples)
- Bundled în /public/samples/

DEPLOY:
- Cloudflare Pages (gratuit, unlimited bandwidth, WASM support)
- COOP/COEP headers pentru SharedArrayBuffer


2. FLUXUL DE DATE AUDIO
================================================================================

File Input (drag/drop) → Main Thread JS (decode audio) → AudioContext (Web Audio)
                                                              │
                                                              ▼
                         ┌──────────────────────────────────────────┐
                         │          AudioWorklet Thread              │
                         │  ┌────────────────────────────────────┐  │
                         │  │     WASM DSP Engine (Rust)          │  │
                         │  │                                      │  │
                         │  │  Input Buffer                        │  │
                         │  │    ↓                                  │  │
                         │  │  [Gate] → [EQ] → [Comp] → [Reverb]  │  │
                         │  │    ↓                                  │  │
                         │  │  Output Buffer                       │  │
                         │  │    ↓                                  │  │
                         │  │  Analysis Data (FFT, RMS, peaks)     │  │
                         │  └────────────────────────────────────┘  │
                         └──────────────┬───────────────────────────┘
                                        │
                              MessagePort (postMessage)
                                        │
                         ┌──────────────▼───────────────────────────┐
                         │          Main Thread                      │
                         │  Zustand Store → Canvas/wavesurfer/       │
                         │                   audiomotion              │
                         └──────────────────────────────────────────┘

Comunicare Main Thread ↔ AudioWorklet ↔ WASM:
- UI trimite parametri via port.postMessage()
- AudioWorklet pasează la WASM via set_param()
- WASM procesează 128 samples/frame
- Analysis data trimisă înapoi la ~30fps (nu la sample rate)

4. RECOMMENDATION ENGINE - LOGICA RULE-BASED
================================================================================

Feature Extraction (Rust WASM):
- RMS loudness (overall, per-band)
- Peak level
- Crest factor (peak/RMS ratio = dynamic range indicator)
- Spectral centroid (brightness)
- Spectral balance (energy per octave band)
- Noise floor estimation (quietest 10% of frames)
- DC offset
- Clipping detection (samples at ±1.0)

Heuristic Rules:
- noise_floor > -40dB → Suggest Noise Gate/Reduction, threshold = noise_floor + 6dB
- crest_factor < 6dB → Warn: already heavily compressed
- crest_factor > 20dB → Suggest compressor: ratio 3:1, threshold at RMS + 6dB
- spectral_energy[60-250Hz] > spectral_energy[250-2kHz] * 1.5 → Boomy/muddy, suggest EQ cut at 200Hz
- spectral_energy[2k-6kHz] > spectral_energy[250-2kHz] * 1.8 → Harsh, suggest EQ cut at 3-4kHz
- spectral_centroid < 1000Hz → Dull, suggest high shelf boost +2dB at 8kHz
- peak_level > -1dB → Suggest limiter at -1dB ceiling
- dc_offset > 0.01 → Suggest HPF at 20Hz


5. EFFECT PARAMETERS
================================================================================

COMPRESSOR:
- threshold (-60 to 0 dB)
- ratio (1:1 to 20:1)
- attack (0.1 to 200 ms)
- release (10 to 1000 ms)
- knee (0 to 30 dB)
- makeup gain (0 to 30 dB)
- sidechain HPF frequency
- dry/wet mix
- auto gain option

GATE:
- threshold (-80 to 0 dB)
- attack (0.01 to 50 ms)
- hold (0 to 500 ms)
- release (5 to 500 ms)
- range (-90 to 0 dB)
- hysteresis (0 to 10 dB)

PARAMETRIC EQ (per band, 4-6 bands):
- frequency (20 to 20000 Hz)
- gain (-24 to +24 dB)
- Q / bandwidth (0.1 to 18)
- filter type (bell, low shelf, high shelf, HPF, LPF, notch)
- slope (6/12/18/24 dB/oct for shelves/passes)

REVERB:
- room size (0 to 1)
- decay time (0.1 to 10 s)
- pre-delay (0 to 200 ms)
- damping (0 to 1)
- wet/dry mix
- early reflections level

DELAY:
- time (1 to 2000 ms)
- feedback (0 to 95%)
- filter (LP/HP on feedback)
- sync mode (free / tempo-synced)
- wet/dry mix
- ping-pong option

LIMITER:
- ceiling (-6 to 0 dB)
- release (1 to 500 ms)
- lookahead (0 to 10 ms)

CHORUS:
- rate (0.1 to 10 Hz)
- depth (0 to 100%)
- voices (1 to 6)
- wet/dry mix

FLANGER:
- rate (0.01 to 10 Hz)
- depth (0 to 100%)
- feedback (-95 to 95%)
- wet/dry mix

PHASER:
- rate (0.01 to 10 Hz)
- depth (0 to 100%)
- stages (2 to 12)
- feedback (-95 to 95%)
- wet/dry mix

SATURATION:
- drive (0 to 30 dB)
- type (tape, tube, transistor, digital clip)
- tone (dark to bright)
- wet/dry mix

DE-ESSER:
- frequency (2000 to 12000 Hz)
- threshold (-40 to 0 dB)
- reduction (0 to 20 dB)
- listen mode (on/off)

TRANSIENT SHAPER:
- attack gain (-24 to +24 dB)
- sustain gain (-24 to +24 dB)
- detection speed

NOISE REDUCTION:
- threshold/sensitivity
- reduction amount (0 to 40 dB)
- smoothing
- learn noise profile

STEREO WIDTH:
- width (0 to 200%)
- mid gain (-12 to +12 dB)
- side gain (-12 to +12 dB)

GAIN/TRIM:
- gain (-inf to +24 dB)
- phase invert
- pan (-100L to 100R)
